## What is Zod?
Zod is a validation library. You describe the shape of your data once, and Zod:

1. Validates incoming data (is this actually an email? is password long enough?)
2. Gives you TypeScript types automatically — no need to write types separately


## What is a DTO?
- DTO = Data Transfer Object. It's just a fancy name for "the shape of data that travels between backend and frontend".
